# Core Tech 矿物与材料系统

本文档记录每次新增矿物、材料和零件后的功能定位。新增矿物不能只注册，必须同步说明现实依据、生成规则、工具等级、风险、掉落、烧炼和用途。

## 总体规则

- 所有 Core Tech 矿石都必须使用铁镐或更高等级工具采掘。
- 所有矿石都加入 `minecraft:mineable/pickaxe` 和 `minecraft:needs_iron_tool`。
- 矿物生成只在主世界地下矿物阶段加入。
- 粗矿可以烧炼或高炉烧炼成对应锭。
- 铀属于辐射材料，不能像普通矿物一样无风险堆在背包里。

## 辐射规则

当前实现：

- 铀矿石拥有轻微发光，便于体现放射性。
- 玩家踩在铀矿石上会获得短暂虚弱效果。
- 粗铀和铀锭使用 `RadioactiveItem`：非创造/旁观玩家背包中持有时，每隔一段时间获得虚弱；堆叠数量大于等于 16 时会额外中毒。

后续计划：

- 加入铅板或防护服降低辐射。
- 铀矿开采需要先用核心工作台制作辐射计或铅衬手套。
- 铀后续进入离心、燃料棒、反应堆多方块结构。

## 第一阶段关联链

1. 采矿获得锡、铝、铅、银、铀等矿物。
2. 粗矿烧炼成锭。
3. 原版铜和铁加工成铜线、铁板、齿轮。
4. 铜线、硅晶圆、红石合成基础电路。
5. 铅锭、铜线、红石合成电池芯。
6. 核心工作台由工作台、铜线、铁板、机械齿轮组合而来，是后续科技路线入口。

## 后续必备系统

- 世界生成平衡测试：确认矿物不会过密或过稀。
- JEI/REI 兼容：展示核心工作台配方和机器配方。
- 机器外壳、线缆、发电机、粉碎机、电炉。
- 铅板/防护服/辐射计，完整化铀的风险收益链。

# 核心工作台系统

核心工作台是整个模组的主入口，不是普通工作台换皮。

## 当前职责

- 可放置、可堆叠。
- 独立方块类：`CoreWorkbenchBlock`。
- 独立方块实体：`CoreWorkbenchBlockEntity`。
- 独立菜单：`CoreWorkbenchMenu`。
- 独立界面：`CoreWorkbenchScreen`。
- 当前提供 3x3 合成区、输出槽、玩家背包区。
- 当前接入原版 crafting recipe 查询。

## 后续职责

- 专属科技配方类型。
- 配方分类：零件、机器、线缆、升级、辐射防护。
- 配方解锁：基础材料 -> 能源 -> 自动化 -> 高能科技。
- 与文档网页联动，展示核心工作台可制作内容。

## UI 规范

- 必须有明显的 3x3 输入区。
- 必须有输出槽和方向引导。
- 玩家背包和快捷栏必须有分区。
- 后续改成纹理 GUI 时，UI 纹理单独存放，不要继续只用 `fill` 画临时界面。

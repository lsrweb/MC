## 建模结论

后续新增机器和管道都需要单独建模，但建模方式不同：

- 普通矿石和简单储物方块：可以使用 `cube_all` 或多面 `cube` 模型。
- 机器方块：必须至少有 front/top/side/back 或方向状态模型，不能只套一个普通方块贴图。
- 管道/线缆：需要连接状态模型，至少区分中心段、六方向连接段、端口、过滤器/阀门等部件。
- 多方块机器：需要独立控制器模型、结构方块模型、完整结构文档和验证逻辑。

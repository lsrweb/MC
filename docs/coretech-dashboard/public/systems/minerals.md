# 矿物与材料系统

## 总体规则

- 所有 Core Tech 矿石必须使用铁镐或更高等级工具。
- 所有矿石都加入 `minecraft:mineable/pickaxe` 和 `minecraft:needs_iron_tool`。
- 每个矿物必须有：矿石方块、粗矿、锭、贴图、模型、语言、掉落表、烧炼配方、世界生成规则、用途说明。
- 新增矿物后必须同步更新本文档和 `docs/coretech_dashboard.html`。

## 辐射规则

- 铀矿轻微发光。
- 玩家踩在铀矿上会获得短暂虚弱。
- 粗铀和铀锭使用 `RadioactiveItem`。
- 非创造/旁观玩家持有粗铀或铀锭时会周期性获得虚弱。
- 堆叠数量大于等于 16 时会额外中毒。
